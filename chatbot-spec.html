4
1 Depth — 대주제 캐러셀 선택

챗봇 최초 진입 시 노출되는 화면. FAQ 카테고리를 캐러셀 카드 형태로 제공하며, 좌우 스와이프 또는 화살표로 탐색한다.

4-1. 캐러셀 카드 구성
카드 번호카드 제목제공 질문 목록
카드 0 자주 묻는 질문 FAQ 목록 air 회원 가입 / 포인트 적립·사용 / 요금 납부·해지 / eSIM·USIM
카드 1 air 회원 가입 air 가입 조건 / T→air 전환 혜택 / 친구초대 혜택
카드 2 포인트 적립·사용 포인트 요금 할인 / 적립·사용 방법 / 포인트샵 교환 상품
카드 3 요금 납부·해지 자동납부 변경 / 납부 내역 확인 / 해지 신청 방법
4-2. 캐러셀 인터랙션
항목명세
카드 폭270px 고정. 우측 약 66px 다음 카드 미리보기(peek) 노출
좌우 화살표첫 카드: 좌 화살표 숨김 / 마지막 카드: 우 화살표 숨김
페이지 인디케이터카드 하단 dot(●) 4개, 현재 위치 강조
우측 페이드우측 48px 그라데이션 오버레이로 추가 카드 존재 암시
슬라이드 단위카드 폭(270) + 간격(10) = 280px
애니메이션cubic-bezier(.4,0,.2,1) 320ms transition
4-3. 카드 내 버튼 동작
동작결과
카드 내 항목 버튼 클릭해당 카테고리·질문으로 2 Depth 이동
카드 0 항목 클릭선택한 대주제의 2 Depth 진입 (질문 목록 표시)
카드 1~3 항목 클릭해당 질문 인덱스를 갖고 2 Depth 진입 후 즉시 3 Depth 이동
4-4. 하단 네비게이션
1 Depth에는 "이전으로 / 처음으로" 버튼 없음. 최초 진입 화면이므로 네비게이션 불필요.
5
2 Depth — 세부주제(질문) 선택

선택된 대주제 카테고리에 속한 FAQ 질문 목록을 하나의 카드 안에 제공한다. 봇 메시지와 선택 버튼이 동일 카드 내에 포함된다.

5-1. 통합 카드 구성
구성요소위치설명
카드 배경전체#e8e8e8 회색 배경, border-radius 14px
봇 아바타카드 헤더 좌측흰색 원형 38×38px (빈 상태)
카테고리 제목카드 헤더 우측선택된 대주제명, font-weight 700
안내 문구헤더 하단카테고리별 고정 안내 텍스트
질문 버튼 목록카드 하단부카테고리별 3개 질문 버튼 (흰색 배경, border)
5-2. 질문 버튼 동작
항목명세
버튼 수카테고리당 3개 고정
버튼 텍스트실제 FAQ 질문 전문 표시
클릭 동작해당 질문 인덱스를 전달하여 3 Depth 이동
hover 상태배경 #f5f5f5, 테두리 #bbb
5-3. 하단 네비게이션 (캡션형)
버튼동작스타일
이전으로1 Depth로 복귀흰색 배경, 회색 테두리, 라운드 버튼
처음으로1 Depth로 복귀 (이전으로와 동일)동일
※ 네비게이션 버튼은 고정 하단 바가 아닌 카드 바로 아래 인라인 배치 (캡션 스타일)
6
3 Depth — 상세 답변 확인

선택한 질문에 대한 상세 답변을 카드 형태로 제공한다. 긴 답변은 더보기로 확장하며, 추가 상담이 필요한 경우 채팅 상담사 연결을 제공한다.

6-1. 상세 카드 구성
구성요소위치설명
질문 제목카드 최상단선택한 질문 텍스트 전문, font-weight 700
답변 목록제목 하단줄 단위 bullet(·) 리스트로 표시
더보기 버튼답변 하단접기/펼치기 토글. 기본 접힘 상태
채팅상담사 연결하기더보기 아래카드 내 최하단 행, 구분선(border-top) 위에 배치
6-2. 채팅상담사 연결하기
항목명세
버튼 위치상세 카드 내부 최하단 (더보기 아래 행)
클릭 동작SKT air 웹채팅 URL 팝업 오픈
팝업 URLhttps://mw-aicc.sktelecom.com/msgfe/chatWrap?popUp=true&id=AIRWEBCH01
팝업 크기width: 400px / height: 700px
오픈 방식window.open(url, '_blank', 'width=400,height=700')
6-3. 더보기 토글
상태표시동작
기본 (접힘)더보기 ∨클릭 시 추가 내용 펼침
펼침접기 ∧클릭 시 초기 상태로 복귀
6-4. 하단 네비게이션 (캡션형)
버튼동작
이전으로2 Depth (질문 목록)으로 복귀
처음으로1 Depth (캐러셀 초기 화면)으로 복귀

FAQ 데이터 명세
7-1. air 회원 가입
질문답변 요약
air 가입 조건이 어떻게 되나요? 19세 이상 대한민국 국적 성인만 가입 가능. 외국인·미성년자 불가. 1인 1회선 제한, 2nd·멀티·테더링·쉐어링 불가.
T에서 air로 전환하면 기존 혜택은 어떻게 되나요? 결합·약정할인 불가, T멤버십 불가, 로밍·데이터 쿠폰 삭제, SKT 장기혜택 상실, 로밍 상품 혜택 미제공 등.
회원가입 시 친구초대 혜택 알려주세요 가입 48시간 내 초대코드 입력 시 초대자·피초대자 각 1,000p 적립. 최대 20명까지 적립 가능.
7-2. 포인트 적립·사용
질문답변 요약
포인트로 요금을 할인받을 수 있나요? 적립 포인트로 요금 할인 가능. 월 최대 5,000포인트 한도.
포인트 적립과 사용 방법을 알려주세요 맞벌기·오늘의 픽·포인트 미션 등 혜택 프로그램으로 적립. 1포인트부터 사용 가능 (air 요금제 가입 고객 한정).
포인트샵에서 어떤 상품으로 교환할 수 있나요? 상품권, 편의점·백화점·식음료 모바일상품권 구매 가능. 브랜드 지속 추가 예정.
7-3. 요금 납부·해지
질문답변 요약
요금 자동납부 방법을 변경하고 싶어요 air App → 전체 → 멤버십 요금안 → 납부 방법 변경 → 변경하기
월별 요금 납부 내역을 확인하고 싶어요 air App → 전체 → 월별 요금 및 이용내역 (최근 3개월 청구금액 제공)
해지는 어떻게 신청하나요? air App 또는 고객센터를 통해 신청. 상세 안내 제공.
